# Asset Requests

The site is complete without these — every visual is generated procedurally.
These assets would upgrade specific moments when available. Fallbacks are
already designed and shipping.

## 1. Pattrix logo (vector)

- **Use**: header, footer, favicon, OG image
- **Current fallback**: letterspaced Archivo wordmark + generated dot-mark favicon
- **Request**: SVG, monochrome-capable, works at 24px height

## 2. Client logos (7)

- **Use**: Clients wall (`content/clients.ts`)
- **Current fallback**: premium text-set names + sector captions
- **Request**: SVG or transparent PNG ≥ 280px wide, mono/gray versions preferred;
  must be cleared for publication by each client

## 3. Case-study photography — one hero image per project

- **Use**: case-study pages (`cover` field) and optionally home work cards
- **Current fallback**: generated signature patterns (rings / rays / interference)
- **Generation/shoot brief** (per project):
  - Subject: real campaign material — set photography, event coverage, or
    production stills from the actual engagement. **No invented/staged client
    content, no stock.**
  - Composition: wide 16:10 (home) and 21:9 (case page) crops from the same
    master; clear focal subject right-of-center or left-of-center (text sits
    opposite); generous negative space
  - Dimensions: master ≥ 2400×1200
  - Lighting: soft, directional, daylight-balanced; avoid mixed color casts
  - Color: neutral-cool grade that harmonizes with `#0171DD`; avoid saturated
    reds/oranges dominating the frame
  - Negative instructions: no visible confidential documents, no unreleased
    faces without consent, no competitor branding
  - Delivery: JPEG/WebP, sRGB, under 600KB after compression
